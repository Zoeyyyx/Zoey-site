# Zoey Site

Zoey Site 是 Zoey 的个人内容站，收纳碎碎念、文章、视觉记录、音乐和工具。

## 打开网站

请使用线上版本访问：

```text
https://zoeyyyx.github.io/Zoey-site/
```

## 信息流框架

- `Home`：首页入口，展示封面、最近更新、站点概览和快速入口。
- `Notes`：碎碎念，短记录按时间线、卡片和关系视图组织。
- `Writing`：文章，放更完整的长文本。
- `Visual`：视觉记录，存放图片、影像和创作片段。
- `Music`：音乐，整理歌、专辑、播放记录和相关想法。
- `Tools`：工具箱，放可交互的小工具或实验页面。
- `Admin`：后台管理入口，用于维护内容与封面图。

站点前端可部署在 GitHub Pages，动态内容和后台数据由 Supabase 提供。
